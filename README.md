### How to run?
clone the codebase
```bash
git clone https://github.com/toantonthat/spring-boot-liquibase.git
```

run
```bash
mvn spring-boot:run
```

Mysql configuration 
  
`JDBC URL`: jdbc:mysql://localhost:3306/liquibase?useSSL=false

`User Name`: root  

`Password`: Leave it blank
